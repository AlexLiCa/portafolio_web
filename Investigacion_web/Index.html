Investigacion HTML,CSS,JS

HTML (Hypertext Markup Language)

1) Es el bloque de construcción más básico de la web
2) Define la estructura del contenido web
3) Hypertext: se refiere a la parte que conecta paginas una con otra ya sea dentro del sitio o con otros sitios
4) Markup: para anotar texto imágenes y otros contenidos para mostrar dentro del buscador web

CSS (Cascading Styles Sheets)

1) Es el código que cambia el estilo de los html
2) No es un lenguaje de programación ni un lenguaje de etiquetado es un lenguaje de hojas de estilo
3) Para utilizar una hoja de estilos dentro de css tienes que referencias la hoja de estilos dentro del encabezado
Ejemplo: link href="styles/MyEstilo.css" rel="stylesheet"

Javascript

1) Es un lenguaje de programación que te permite implementar características complejas a tus paginas web
2) Siempre que una pagina web hace mas que solo mostrar información es por que es una pagina de javascript
